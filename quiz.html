Quick Reference Formulas

Time Dilation
Δt = Δt₀/√(1 - v²/c²)
Δt₀ = proper time
Length Contraction
L = L₀√(1 - v²/c²)
L₀ = proper length
Lorentz Transformation
x' = γ(x - vt)
t' = γ(t - vx/c²)
γ = 1/√(1 - v²/c²)
Spacetime Interval
Δs² = Δx² - c²Δt²
• Δs² < 0: timelike
• Δs² > 0: spacelike
• Δs² = 0: lightlike
